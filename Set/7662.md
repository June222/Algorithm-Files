### 7662 - 이중 우선순위 큐

# 사용된 알고리즘 :
	1. 자료구조(multiSet)

# 해결 목표 :
	1. Data를 넣거나 삭제할 수 있다.
	2. 최소값, 최대값을 정하여 삭제할 수 있다.

# 해결해야 할 부분 :

	1. 최소값, 최대값을 정하여 삭제할 수 있다.

	-> 기존에 알고 있던 Heap은 최대와 최소를 동시에 정하지는 못한다.
	-> 다른 자료구조가 필요하다.

	-> Set은 Data를 입력함과 동시에 오름차순으로 정렬한다.
	-> 그리고 Key값으로 특정 Data를 삭제할 수 있다.

	-> 다만, Data는 중복되지 못한다.


	2. Data는 중복가능하고, 삭제할 때는 하나만 삭제되어야한다.

	-> 이 부분이 multiSet과 Set의 차이점이다.
	-> multiSet은 Set을 그대로 가지고 오나 "Data가 중복 가능하다." 라는 차이점을 가진다.

	-> 또한 삭제할 때에서도 차이를 가진다.
	-> 1) ms.erase(Key)를 하게 되면 Key값을 가진 모든 Data를 삭제한다라는 뜻이다.
	-> 2) ms.erase(&Key)는 주소지의 Data 하나만을 지운다.

	-> 따라서 이렇게 활용가능하다.
	-> ms.erase(ms.begin()); ms.erase(ms.end()-1);

	-> ms.end()는 "마지막 데이터의 주소 + 1" 를 가르키고 있어, -1로 해주어야한다.
