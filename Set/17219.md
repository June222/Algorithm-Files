### 17219 - 비밀번호 찾기

# 사용된 알고리즘 :
	1. 자료구조 (map)

# 해결 목표 :
	1. site와 비밀번호가 동시에 입력된다. (10만개)
	2. 비밀번호를 찾고싶은 site가 입력되면, 비밀번호를 출력한다. (무조건 존재하는 사이트)

# 해결해야 할 부분 :
	1. map STL에 대해서 공부한다.
	-> map은 배열과 같이 사용가능하다.
	-> 1) "map <key_type, value_type> m;"으로 선언한다.
	-> 2) "map[key] = value"로 저장이 가능하다.
	-> 3) "map[key]"는 value를 return 한다.
