### 14425 - 문자열 집합

# 사용된 알고리즘 :
	1. 자료구조(Set)

# 해결 목표 :
	1. 집합에 문자열이 있는데 입력 받은 문자열들 중에서 몇 개의 문자열이 집합에 있는지 출력하시오.

# 해결해야 할 부분 :
	1. Set의 사용방법
	-> Set에 저장하는 방법과 Data가 Set에 있는지 확인하는 방법
	-> 1) "Set <data_type> s"로 선언한다.
	-> 2) "s.insert(data)"로 저장한다.

	* 중요 *
	-> 3) "s.find(data)"는 반복자를 return하는데,
	-> 3-1) 찾은 경우에는 위치를,
	-> 3-2) data가 없는 경우에는 s.end()를 return한다.

	-> 4) 따라서 찾은 경우만 쓰고 싶다면 "if(iter!=s.end())"를 쓰면 된다.
