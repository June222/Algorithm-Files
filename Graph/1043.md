### 1043 - 거짓말

# 사용된 알고리즘 :
	1. 그래프 이론
	2. 자료 구조
	3. 그래프 탐색
	4. 분리 집합

# 해결 목표 :
	1. 거짓말 할 수 있는 파티의 최대 수를 구하여라.
	2. 진실을 알고 있는 자의 파티에서는 진실만 말할 수 있다.
	3. 어떤 파티에서는 진실을, 다른 파티에서는 거짓을 들은 자는 거짓말을 알아차린다.

# 해결해야 할 부분 :
	1. 순서대로 확인해서는 안된다.
	-> 예를 들어 1, 2 / 7, 8 / 5, 7 순으로 파티에 참여했다고 가정하자.
	-> 1과 5가 진실을 아는 자라면 거짓말 할 수 있는 파티 수는 0개이다.
	-> 왜냐하면 7이 거짓을 들으면 안되기 때문.
	-> 그런데 순서대로 검사하면 1이 나온다.
	-> 이것을 막으려면 어떻게 해야하나.

	-> 모든 경우에 대해서 재귀적으로 검사해야한다.
	
	-> 1) 진실을 아는 자가 있는 파티는 거짓말 할 수 없다.
	-> 2) "그 자가 있는 모든 파티"의 "모든 사람들" 에게 1)을 반복한다.
