### 17626 - Four Squares

# 사용된 알고리즘 :
	1. Dynamic Programming

# 해결 목표 :
	1. 5만개의 자료와 0.5초의 시간
	2. 세상 모든 수는 그보다 작은 4개 이하의 수의 제곱의 합으로 표현 가능할 때, 최소의 개수를 구하라.

# 해결해야 할 부분 :
	1. 0.5초..
	-> 1억번의 계산 -- 1초
	-> 0.5초 -- 5천만번의 계산
	-> 최대 5만개의 자료는 하나의 자료당 1천번 이하의 계산으로 끝내야함.
	
	-> 1) 5만의 제곱근은 223
	-> 2) 1 ~ 223 으로 모두 확인하여도 시간은 충분함.

	2. 따라서 DP
	-> Answer[N] = Answer[a] + Answer[N-a];
	-> bottom-up으로 하면 됨.
	-> 구현 부분을 정확히 생각해 볼것
