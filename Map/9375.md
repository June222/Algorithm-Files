# 사용된 알고리즘 :

# 해결 목표 :
	1. 매일 옷을 겹치지 않게 적어도 하나는 걸쳐야 할 때, 조합의 수를 구하시오.

# 해결해야 할 부분 :
	1. Map의 사용방법
	-> "Map<key_type, value_type> map"으로 선언한다.
	-> Map의 Key값에는 string 또한 들어갈 수 있다.

	* 중요 *
	-> iter 사용시에는 for문을 다음과 같이 변경할 수 있다.
	ex) for(auto iter=map.begin(); iter!=map.end(); iter++) == for(auto iter : map)

	-> 또한 이렇게 선언했을 시 begin()을 가르키는 것과 달리 원소 자체가 iter가 된다.
	-> 그래서 (*iter)라고 할 필요 없다!!!!!!!!!!!!!!!


	2. 조합론
	-> 단순 수학문제다.
	-> 또한 이 문제는 의상의 이름은 필요하지 않다. 총 개수만 필요하다.
	-> 그리고 입거나 안 입거나로 경우의 수를 구할 수 있다.
	-> 다만 모두 안 입는 경우는 빼야한다.
