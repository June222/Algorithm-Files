### 1541 - 잃어버린 괄호

# 사용된 알고리즘 :
	1. String
	2. 구현

# 해결 목표 :
	1. 숫자와 '+', '-'만으로 이루어진 문자열에 괄호를 넣어 계산식의 결과가 최소가 되게 하여라.
	2. 문자열을 int로 변환하라.

# 해결해야 할 부분 :
	1. 괄호를 직접 문자열에 넣어야만 하는 가?
	-> 구현 부분에 있어 아직 잘 정립이 되어있지 않은 부분이다.
	-> 꼭 하란대로 구현할 필요없다.
	-> 기능을 정확히 수행하고 오류가 없으면 된다.

	-> 문자열은 숫자로 얼마든지 바꿀 수 있다. 이 부분은 다음 알고리즘에선 고려할 필요가 없다.
	-> 1) 어떻게 하면 최소가 되는가?
	-> 2) 뺄셈이 등장하면 뒤에있는 모든 수의 뺄셈을 진행하면 된다.
	-> 3) 즉, 뺄셈이 등장하기 전 까지의 숫자만 더할 수 있다.
