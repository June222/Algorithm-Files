### 1107 - 리모컨

# 사용된 알고리즘 :
	1. DFS

# 해결 목표 :
	1. 주어진 채널에 부서지지 않은 숫자패드로 가장 가까운 숫자를 만들어야한다.
	2. 꼭 같은 자리수만 가장 빠른 지점인가?
	3. 언제 DFS를 빠져 나갈 것인가?

# 해결해야 할 부분 :
	1. 항상 시간 복잡도를 계산.
	-> 최악의 경우 9^7 이고, 이는 약 54만번의 계산(0.05초)으로 할 수 있으므로 DFS로 찾는다.
	
	2.  9999에 버튼은 1만 되는 경우, 1000에 9만 되는 경우
	-> 1) 9999-1111 = 약 8천번, 11111-9999 = 약 2천번
	-> 자리수가 더 큰 수가 더 횟수가 적을 수 있다.

	-> 2) 1000-999 = 1번, 9999-1000 = 약 9천번
	-> 자리수가 더 작은 수가 더 횟수가 적을 수 있다.

	3. 함수의 종료시점을 명확히 정해야한다. 무한루프 방지.
	-> 처음에는 같은 자리 수까지 숫자를 키우면 빠져나간다고 하였다.
	-> 2-2.의 경우에 모든 경우를 다 살펴봐야한다.
	-> 또한 시간복잡도에 의하면 7자리까지 하여도 충분한 시간을 가진다.
	-> 자리수 + 2 가 되었을 때 종료한다.
